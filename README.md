Aplicação que gerencia locações de robôs industriais, agrícolas e domésticos para clientes empresariais e individuais. 
Nesse projeto, é implementada uma GUI com Java Swing, tratamento de eventos, bem como leitura de arquivos .csv
O foco do programa é aplicar os conhecimentos adquiridos na faculdade.

Obs.: A aplicação não possui todas as funcionalidades requisitadas inicialmente, e até o momento não está completa.
